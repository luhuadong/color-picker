# Color Picker - 屏幕颜色拾取工具

一个纯网页实现的屏幕颜色拾取工具，可以在屏幕上任意位置拾取颜色值。

## 功能特点

- 🎨 **全屏颜色拾取**：使用 `getDisplayMedia()` API 捕获屏幕，可以在屏幕上拾取颜色
- 🔍 **放大镜效果**：10倍放大，方便精确选择像素
- 📋 **多种颜色格式**：支持 RGB、HEX、HSL 三种颜色格式显示
- 📱 **一键复制**：快速复制 HEX 颜色值
- 🌐 **纯网页实现**：无需安装任何软件或扩展，直接在浏览器中使用

## 使用方法

1. 点击 "Pick" 按钮
2. 浏览器会请求屏幕共享权限，选择要捕获的屏幕或窗口
3. 移动鼠标，放大镜会显示当前区域的放大效果
4. 点击鼠标确认拾取颜色
5. 按 ESC 键可取消拾取

## 技术实现

- 使用 `getDisplayMedia()` API 捕获屏幕内容
- Canvas API 处理图像数据并提取像素颜色
- 纯 HTML/CSS/JavaScript 实现，无依赖

## 局限性说明

**⚠️ 纯网页方案的技术限制：**

由于浏览器的安全策略限制，纯网页应用存在以下局限性：

1. **仅在浏览器窗口内工作**：拾取笔（放大镜）只能在浏览器窗口范围内移动，无法脱离浏览器窗口
2. **需要屏幕共享权限**：需要用户授权屏幕共享，并且会显示共享指示器
3. **受浏览器限制**：不同浏览器对屏幕捕获的支持和表现可能不同

这些限制是浏览器的安全机制，旨在防止恶意网站访问用户的屏幕内容。

## 更好的体验

如果需要更好的使用体验，建议使用浏览器扩展（插件）：

- **GetColor**：专业的屏幕取色器扩展，支持全局拾取、跨窗口操作、更多颜色格式等功能
- 其他浏览器扩展可以在 Chrome Web Store、Firefox Add-ons 等扩展商店中搜索 "color picker" 或 "screen color"

浏览器扩展可以获得系统级权限，实现真正的全局颜色拾取功能。
